La ligne de commande sous Linux
Fonctionnalités Commande à effectuer
Afficher un manuel système man
Afficher le contenu du dossier actuel ls
Afficher le contenu du dossier actuel avec les fichiers cachés ls -a
Afficher le contenu du dossier courant version longue ls -l
Afficher le contenu du dossier courant version longue avec les fichiers cachés ls -al
Afficher le chemin du dossier actuel pwd
Se déplacer dans un dossier cd nom du dossier
Revenir à un dossier précédent cd ..
Se mettre en mode super utilisateur (à utiliser avec modération !) sudo -s
Sortir du mode super utilisateur exit
Télécharger les mises à jour des paquets apt update
Mettre à jour les paquets apt upgrade
Trouver tous les paquets contenant un mot spécifique search nom du paquet
Installer un paquet apt install nom du paquet
Installation d'un fichier .deb dpkg -i nomdufichier.deb
Créer un dossier mkdir nom du dossier
Créer un dossier et un sous dossier en une fois mkdir -p nom du dossier/nom du sous dossier
Créer un fichier touch nom du fichier
Ouvrir un fichier avec un logiciel spécifique nom du logiciel + nom du fichier
Déplacer un fichier ou dossier mv nom du fichier suivi du nom du dossier
Renommer un fichier ou dossier mv nom du fichier suivi du nouveau nom
